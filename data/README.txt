ICISO Application Data Directory
===============================

This directory stores all volunteer and internship applications submitted through the platform.

Files:
- applications.json: Main application database (auto-created)
- applications-export-*.csv: Exported application reports

Access Methods:
1. Admin Dashboard: http://localhost:3000/admin
2. Terminal: npm run view-applications
3. Export: npm run export-applications

All applications are automatically sent to: icisoi.club@gmail.com

Alhamdulillah for serving the Muslim community!
